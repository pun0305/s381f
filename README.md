## Installation

Clone the repository and run `npm install`

```
git clone https://github.com/g1153567/s381f-b.git
npm install
npm start
```

## Push to git

```
rm -rf .git
git init
git add .
git commit -a -m'1st commit'
git remote add origin <your repo>
git push -f origin master
```

## Starting the server

```
npm start
```
